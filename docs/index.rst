Welcome to goopylib's documentation!
====================================

goopylib is a powerful graphics and application development library for Python. It is built in C++ using OpenGL and provides a Python Interface to allow for the quick creation of beautiful user interfaces and games.

Get started by :ref:`installing<installation>` and following the :ref:`user guide<user-guide>`.
