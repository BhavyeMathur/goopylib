The goopylib library is powerful game, graphics, and GUI API for Python and C++ to build cross-platform applications.

It allows you to unlock the potential of OpenGL and low-level graphics with a high-level, simple-yet-powerful interface.

Find the source code on `GitHub <https://github.com/BhavyeMathur/goopylib>`_.
